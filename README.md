
### Installation

To setup the project for the first time, please follow below steps:

1. Clone the repo
   ```sh
   git clone https://github.com/NSWDAC/CCE-Qlik-Mashup.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Initiate dev server
   ```js
   yarn start
   ```
4. Create a production build
   ```js
   yarn build
   ```
